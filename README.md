# BCTA

Bhimavaram Cell Phone Technicians Association - Management Portal

## Features

- Member Management
- Meeting Management with QR Attendance
- Payment Tracking
- Complaint Management
- Notifications System

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Firebase (Auth, Firestore, Storage)
- Recharts for analytics

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
